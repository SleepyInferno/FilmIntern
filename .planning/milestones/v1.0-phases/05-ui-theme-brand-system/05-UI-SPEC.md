## Color

### Theme: Dark Mode (default)

| Role | Value | Usage |
|------|-------|-------|
| Dominant (60%) | `oklch(0.145 0 0)` | Page background (`--background`) |
| Secondary (30%) | `oklch(0.205 0 0)` | Cards, popover, sidebar (`--card`) |
| Accent / Primary (10%) | `oklch(0.769 0.188 70.08)` | See reserved-for list below |
| Destructive | `oklch(0.704 0.191 22.216)` | Delete actions, error states only |

### Theme: Light Mode

| Role | Value | Usage |
|------|-------|-------|
| Dominant (60%) | `oklch(1 0 0)` | Page background (`--background`) |
| Secondary (30%) | `oklch(0.97 0 0)` | Cards, sidebar, muted surfaces |
| Accent / Primary (10%) | `oklch(0.666 0.179 58.318)` | See reserved-for list below |
| Destructive | `oklch(0.577 0.245 27.325)` | Delete actions, error states only |

### Primary Token Values (replaces current grayscale `--primary`)

| Theme | `--primary` | `--primary-foreground` |
|-------|-------------|----------------------|
| Dark | `oklch(0.769 0.188 70.08)` (amber-500) | `oklch(0.145 0 0)` (near-black) |
| Light | `oklch(0.666 0.179 58.318)` (amber-600) | `oklch(1 0 0)` (white) |

### Accent Color Presets

Each preset replaces `--primary` and `--primary-foreground` in both `:root` and `.dark`. All OKLCH values, muted/tasteful saturation.

| Preset | Dark `--primary` | Light `--primary` | localStorage key |
|--------|-----------------|-------------------|-----------------|
| Orange/Amber (default) | `oklch(0.769 0.188 70.08)` | `oklch(0.666 0.179 58.318)` | `accent-color: amber` |
| Electric Blue | `oklch(0.7 0.15 250)` | `oklch(0.55 0.18 250)` | `accent-color: blue` |
| Emerald Green | `oklch(0.72 0.17 160)` | `oklch(0.6 0.18 160)` | `accent-color: emerald` |
| Purple | `oklch(0.7 0.15 300)` | `oklch(0.55 0.17 300)` | `accent-color: purple` |

All presets use `--primary-foreground: oklch(0.145 0 0)` in dark mode and `oklch(1 0 0)` in light mode.

### Accent Reserved For

Accent (the `--primary` token, amber by default) is reserved for:

1. Primary action buttons (Run Analysis, Generate, Save Settings)
2. Active navigation indicator in top nav (border-bottom + text color)
3. Focus rings on interactive elements (via `--ring`)
4. Link and icon hover states
5. Active tab indicators
6. Settings gear icon when on /settings page

Accent must NOT be used for: body text, card backgrounds, borders (except active nav indicator), section backgrounds.

### Card Elevation

| Theme | Border | Shadow |
|-------|--------|--------|
| Dark | `1px solid` using `--border` (`oklch(1 0 0 / 10%)`) | `0 1px 3px oklch(0 0 0 / 0.3)` |
| Light | `1px solid` using `--border` (`oklch(0.922 0 0)`) | `0 1px 3px oklch(0 0 0 / 0.08)` |

### Top Nav Theme Adaptation

| Theme | Background | Border | Text inactive | Text active |
|-------|-----------|--------|---------------|-------------|
| Dark | `oklch(0.145 0 0)` (match page bg) or `oklch(0.12 0 0)` (slightly darker) | `--border` | `--muted-foreground` | `--primary` |
| Light | `oklch(1 0 0)` (white) | `--border` | `--muted-foreground` | `--primary` |

Current hardcoded `bg-stone-950`, `text-stone-400`, `text-amber-500`, `border-stone-800` must be replaced with CSS variable references so both themes work.

## Interaction Contract

### Theme Toggle

| Property | Value |
|----------|-------|
| Location | Top nav, right side, left of settings gear icon |
| Element | `<button>` with aria-label |
| Icon (dark mode) | `Moon` from lucide-react, size 18 |
| Icon (light mode) | `Sun` from lucide-react, size 18 |
| Transition on toggle | None (instant class swap, per user decision) |
| Persistence | `localStorage` key: `theme` with values `dark` or `light` |
| Default | `dark` |
| Mechanism | Toggle `.dark` class on `<html>` element via `next-themes` or equivalent provider |
| Flash prevention | Script in `<head>` reads `localStorage` and sets `.dark` class before paint |

### Accent Color Picker

| Property | Value |
|----------|-------|
| Location | Settings page (`/settings`), new section below existing AI provider settings |
| Layout | Row of 4 circular swatches, 32px diameter, 8px gap between swatches |
| Swatch appearance | Filled circle with the preset's dark-mode `--primary` color |
| Selected state | 2px ring offset with `--primary` color + small checkmark icon (Check from lucide, size 14, centered) |
| Hover state | Scale to 1.1x with `transform` |
| Persistence | `localStorage` key: `accent-color` with values `amber`, `blue`, `emerald`, `purple` |
| Default | `amber` |
| Effect | Immediately updates `--primary` and `--primary-foreground` CSS variables on `<html>` style attribute |
| Save behavior | Client-side only (localStorage), no server round-trip needed |

### Provider Integration

| Component | Integration Point |
|-----------|-------------------|
| `src/app/providers.tsx` | Wrap children with `ThemeProvider` from `next-themes` (attribute: `class`, defaultTheme: `dark`, storageKey: `theme`) |
| `src/app/layout.tsx` | Remove hardcoded `className="dark"` from `<html>`, add `suppressHydrationWarning` |
| `src/app/globals.css` | Update `:root` `--primary` to amber-600, `.dark` `--primary` to amber-500; update `--ring` to match `--primary` in both themes |
| `src/components/app-topnav.tsx` | Replace all hardcoded `stone-*` and `amber-*` classes with CSS variable equivalents; add theme toggle button |
| `src/app/settings/page.tsx` | Add accent color picker section below AI provider card |
| `src/components/ui/card.tsx` | Add `shadow-sm` class for subtle elevation |
