- **better-sqlite3 over Prisma** ‚Äî Prisma 7 dropped direct DB connections, requires driver adapters; better-sqlite3 simpler for local tool
- **Sidebar in root layout** ‚Äî always visible across all pages
- **Native `<select>` for dropdown** ‚Äî no shadcn Select needed; simpler and accessible
- **Auto-save on analysis complete** ‚Äî not on every keystroke; project created immediately on file upload so it appears in sidebar right away
- **`projectIdRef` pattern** ‚Äî avoids stale closure issues when `currentProjectId` is set inside async flow
- **Title from filename** ‚Äî "my_script.pdf" ‚Üí "My Script"; no user prompt at creation time
- **Reset `currentProjectId` on new file upload** ‚Äî ensures new project created for each upload
