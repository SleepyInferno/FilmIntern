## Spacing Scale

Declared values (must be multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps, badge internal padding, inline spacing |
| sm | 8px | Compact element spacing, card content gaps |
| md | 16px | Default card padding (px-4), card grid gap on mobile |
| lg | 24px | Card grid gap on desktop, workspace header padding |
| xl | 32px | Space between workspace header and card grid |
| 2xl | 48px | Not used this phase |
| 3xl | 64px | Not used this phase |

Exceptions: Touch target for card header collapse toggle is 44px minimum height (accessibility).

## Typography

| Role | Size | Weight | Line Height |
|------|------|--------|-------------|
| Body | 14px (text-sm) | 400 (regular) | 1.43 (20px) |
| Label | 12px (text-xs) | 600 (semibold) | 1.33 (16px) |
| Heading | 16px (text-base) | 500 (medium) | 1.375 (leading-snug) |
| Display | 20px (text-xl) | 600 (semibold) | 1.2 |

**Role mapping for this phase:**

- **Display**: Workspace title ("Story Lab Workspace")
- **Heading**: Card titles (CardTitle component, already text-base font-medium)
- **Label**: Badge text, status bar text, "Show more" toggle, score label
- **Body**: All card content text (quotes, descriptions, beat details, theme explanations)

## Layout Contract

### Card Grid

- Container: CSS Grid with `grid-cols-1 md:grid-cols-2 xl:grid-cols-3`
- Gap: 24px (`gap-6`)
- Cards span full column width (no fixed pixel widths)
- Cards that contain significantly more content (e.g., "Story Structure" with many beats) may use `md:col-span-2` to span two columns

### Workspace Header

- Full width above the card grid
- Layout: flex row with items-center, gap-3
- Elements left-to-right: workspace title (text-xl semibold) + project type chip (Badge default variant) + score badge (ScoreBadge with accent bg)
- Bottom margin: 32px (`mb-8`) separating header from grid

### Streaming Status Bar

- Full width, between workspace header and card grid
- Background: `bg-muted` with rounded-lg
- Padding: 8px 16px (`py-2 px-4`)
- Content: pulsing accent dot (8px circle, `animate-pulse bg-primary`) + status text (text-sm text-muted-foreground)
- Disappears entirely when streaming completes (conditional render, not hidden)

### EvaluationCard (Collapsible)

- Uses existing Card component as wrapper
- CardHeader contains: CardTitle (left) + CardAction with ChevronDown/ChevronUp icon button (right)
- Clicking anywhere on CardHeader toggles collapsed/expanded state
- CardContent hidden when collapsed (height 0 with overflow-hidden)
- All cards start expanded by default
- Collapse state is component-local (useState), not persisted
- CardHeader cursor: `cursor-pointer`
- Transition: match accordion animation timing from existing accordion component

### Card Content Patterns

**List items with badges** (e.g., Key Quotes, Soundbites):
- Each item: flex column, quote text (body), then flex row with speaker label + usefulness badge
- Items separated by Separator component or `divide-y`

**Beat/row items** (e.g., Story Structure):
- Each row: flex row with items-center, gap-2
- Elements: beat name (body, flex-1) + page position label (text-xs text-muted-foreground) + effectiveness badge
- Row is clickable (`cursor-pointer hover:bg-muted/50 rounded-md px-2 py-1.5`)
- Clicking expands inline description below the row (accordion-style)

**Dense nested data** (e.g., Character Arcs):
- Show top 3 items by default
- ShowMoreToggle below: "Show N more" / "Show less"
- Toggle text: text-xs text-primary font-medium, with underline on hover
- Toggle uses ChevronDown/ChevronUp icon (12px) inline

## Interaction Contract

### Card Collapse/Expand

| Trigger | Action |
|---------|--------|
| Click card header | Toggle card content visibility |
| Chevron icon | Rotates to indicate state (down=collapsed, up=expanded) |
| Default state | All cards expanded |
| Animation | Height transition matching accordion timing |

### Beat Row Expand (Narrative only)

| Trigger | Action |
|---------|--------|
| Click beat row | Expand/collapse inline description text below |
| Visual affordance | Hover background change on row |
| Icon | Small ChevronRight rotates to ChevronDown when expanded |

### Show More/Less Toggle

| Trigger | Action |
|---------|--------|
| Click "Show N more" | Reveal remaining items beyond initial 3 |
| Click "Show less" | Collapse back to 3 items |
| Count display | Dynamic: "Show 4 more" when 7 total items |

### Streaming Progressive Reveal

| State | Visual |
|-------|--------|
| Card data not yet arrived | Skeleton card: Card shell with 3 Skeleton lines inside (h-4 widths: 100%, 75%, 50%) |
| Card data partially arrived | Real card renders with available data |
| Card data complete | Full card, no skeleton |
| All streaming complete | Status bar disappears |
