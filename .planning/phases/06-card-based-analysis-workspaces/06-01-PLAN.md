- When not collapsed: if `ready` is true, render `children` inside `<CardContent>`; if `ready` is false, render `<CardContent>` with 3 Skeleton lines (h-4, widths: w-full, w-3/4, w-1/2)
- When collapsed: do not render CardContent at all
- Use `overflow-hidden` transition on CardContent wrapper for smooth collapse animation
- Export: `export function EvaluationCard({ title, ready, children, className }: EvaluationCardProps)`

**6. `streaming-status-bar.tsx`** -- Status indicator during streaming:

  - `<span className="text-sm text-muted-foreground">{currentSection}</span>`
- Export: `export function StreamingStatusBar({ currentSection }: { currentSection: string | null })`

**7. `show-more-toggle.tsx`** -- Expand/collapse for dense lists:

- When expanded: `<button ...>Show less <ChevronUp className="h-3 w-3" /></button>`
- Export: `export function ShowMoreToggle({ totalCount, visibleCount, expanded, onToggle }: ShowMoreToggleProps)`

All components must be 'use client' and import from lucide-react for ChevronDown/ChevronUp.
