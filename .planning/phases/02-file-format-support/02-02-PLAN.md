Required contract after this plan:
```typescript
export interface StructuredElement {
  type:
    | 'scene-heading'
    | 'character'
    | 'dialogue'
    | 'parenthetical'
    | 'action'
    | 'transition'
    | 'paragraph';
  text: string;
}

export interface ParseResult {
  text: string;
  metadata: {
    wordCount: number;
    lineCount: number;
    format: string;
    filename?: string;
    size?: number;
    structure?: {
      kind: 'screenplay' | 'document';
      elements: StructuredElement[];
    };
  };
}
```

Current preview:
```tsx
<pre>{text}</pre>
```
This plan must add structure-aware rendering while keeping the raw-text fallback.
