    1. Parse request body: `const { text, projectType } = await req.json()`

    4. Call `streamText` from 'ai':
    ```typescript
    import { streamText, Output } from 'ai';
    import { anthropic } from '@ai-sdk/anthropic';
    import { documentaryAnalysisSchema } from '@/lib/ai/schemas/documentary';
    import { documentarySystemPrompt } from '@/lib/ai/prompts/documentary';

    const result = streamText({
      model: anthropic('claude-sonnet-4-5'),
      output: Output.object({ schema: documentaryAnalysisSchema }),
      system: documentarySystemPrompt,
      prompt: `Analyze this interview transcript for documentary filmmaking purposes:\n\n${text}`,
      providerOptions: {
        anthropic: { structuredOutputMode: 'auto' },
      },
      onError({ error }) {
        console.error('Analysis streaming error:', error);
      },
    });
    return result.toTextStreamResponse();
    ```

    **IMPORTANT:** The AI SDK 6 `toTextStreamResponse()` sends the structured output as a text stream. The client must parse partial JSON from this stream. See client-side consumption in Task 2.

    Wire the "Run Analysis" button (already present from Plan 02):
    - onClick handler `handleAnalyze`:
      1. Set `isAnalyzing = true`, `analysisError = null`, `analysisData = null`
      2. Call `fetch('/api/analyze', { method: 'POST', headers: { 'Content-Type': 'application/json' }, body: JSON.stringify({ text: uploadData.text, projectType: 'documentary' }) })`
      3. If response not ok: set `analysisError = 'Analysis could not be completed. Check your connection and try again.'`, set `isAnalyzing = false`, return
      4. Read the streaming response. The AI SDK `toTextStreamResponse()` sends the structured object as progressive text. Consume using:
      ```typescript
      const reader = response.body?.getReader();
      const decoder = new TextDecoder();
      let accumulated = '';

      while (reader) {
        const { done, value } = await reader.read();
        if (done) break;
        accumulated += decoder.decode(value, { stream: true });
        try {
          const partial = JSON.parse(accumulated);
          setAnalysisData(partial);
        } catch {
          // Incomplete JSON — continue accumulating
        }
      }
      setIsAnalyzing(false);
      ```
      NOTE: This is a pragmatic approach. The stream may send partial JSON that fails to parse — that is expected. Only update state when a valid partial parse succeeds. If the AI SDK sends text in a different format (e.g., prefixed data), adapt the parsing accordingly. The key contract: eventually `accumulated` contains the full JSON object matching DocumentaryAnalysis.

      5. Wrap the entire fetch in try/catch. On catch: set `analysisError = 'Analysis could not be completed. Check your connection and try again.'`, set `isAnalyzing = false`.
