Add an accent color picker section to the settings page. This is a CLIENT-ONLY feature using localStorage -- it does NOT integrate with the existing AI settings API save flow.

1. Import from theme library:
   ```typescript
   import { ACCENT_PRESETS, type AccentColor, applyAccentColor, getStoredAccent, setStoredAccent } from '@/lib/theme';
   import { useTheme } from 'next-themes';
   import { Check } from 'lucide-react';
   ```

2. Add accent color state inside `SettingsPage`:
   ```typescript
   const { theme } = useTheme();
   const [accent, setAccent] = useState<AccentColor>('amber');
   useEffect(() => { setAccent(getStoredAccent()); }, []);
   ```

3. Create a handler function:
   ```typescript
   function handleAccentChange(color: AccentColor) {
     setAccent(color);
     setStoredAccent(color);
     applyAccentColor(color, (theme === 'dark' ? 'dark' : 'light'));
   }
   ```

4. Add a NEW `<Card>` section BELOW the existing AI provider Card (after the closing `</Card>` of the AI settings). Use a `<div className="mt-6">` wrapper for spacing.

5. The accent card content:
   - Section heading: `<h2 className="text-sm font-semibold mb-1">Accent Color</h2>`
   - Description: `<p className="text-xs text-muted-foreground mb-4">Choose the accent color used across the app</p>`
   - Swatch row: `<div className="flex gap-2">` containing 4 buttons

6. Each swatch button (for each key in ACCENT_PRESETS):
   ```tsx
   <button
     key={color}
     onClick={() => handleAccentChange(color)}
     className="w-8 h-8 rounded-full flex items-center justify-center transition-transform hover:scale-110 ring-offset-background"
     style={{ backgroundColor: ACCENT_PRESETS[color].dark.primary }}
     aria-label={`${color} accent color`}
   >
     {accent === color && <Check size={14} className="text-white" />}
   </button>
   ```
   The swatch always shows the dark-mode primary color for visual consistency regardless of current theme.

7. Selected swatch gets an additional ring: add conditional class `${accent === color ? 'ring-2 ring-primary ring-offset-2' : ''}` to the button className.

8. The swatch labels for the 4 colors (in order): `amber`, `blue`, `emerald`, `purple`. Display names as capitalized labels below or use aria-label only (aria-label is sufficient per UI-SPEC).
  </action>
